# Java-StudentReportCard# Student Report Card – Java Console App

This Java program:
- Takes student details
- Accepts marks for multiple subjects
- Calculates total, average, and grade
- Displays formatted report card

## Technologies Used
- Java (Console Program)
- Scanner class for input
